Hướng dẫn
Cách lấy OpenRouter API Key:
  1. Truy cập OpenRouter.ai
  2. Đăng ký tài khoản hoặc đăng nhập
  3. Vào API Keys trong dashboard
  4. Click Create Key và đặt tên
  5. Copy API key và paste vào trường trên
Lưu ý: Free models có giới hạn request. Paid models cần nạp credit vào tài khoản OpenRouter.
